pofview

pofview does one thing: it opens FreeSpace 2 POF (.pof) models and lets you look around. Point it at a model and a standalone POF library loads the geometry; Qt Quick 3D turns it into a shaded surface you can orbit. It is small, Linux-only, and deliberately narrow: no editing, no importers or exporters, just a clean way to open a .pof and see what is inside.

Behind that small remit is a long lineage. pofview descends from the FreeSpace community’s POF Constructor Suite 2, the editor used to build many of these models. It sets aside the editor’s UI and toolchain, keeps its file loader, extracts it into a standalone library, and gradually brings the code into modern C++. The difficult archaeology — understanding the format — was done long ago by others; this is a quieter, read-only reframing of that work, kept faithful to the models it was always meant to open.
